## Implemented Puzzles

### Day 3: No Matter How You Slice It
**Link**: [https://adventofcode.com/2018/day/3](https://adventofcode.com/2018/day/3)

**Description**: Determine overlapping fabric claims and identify the non-overlapping claim. Elves are cutting fabric for Santa's suit and need to find which areas have conflicts.

**Source Code**: [NoMatterHowYouSliceItAOC2018Day3.java](day3/)

**Algorithm**: Track fabric coverage using a coordinate map to count overlaps, then identify the single claim without any overlaps.
